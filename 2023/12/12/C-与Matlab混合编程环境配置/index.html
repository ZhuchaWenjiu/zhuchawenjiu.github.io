复现论文 Developability-Driven Piecewise Approximations for Triangular Meshes 环境需要

添加库目录和链接目录

  • 链接目录

    E:\Matlab\extern\lib\win64\microsoft

  • 库目录

    E:\Matlab\extern\include

这两步做完就可用调用 Matlab 的库函数了，但是运行还是会报错但是已经可用把 Matlab 的命令行调出来了如果调不出来可能是环境变量的问题，把 Matlab 添加到环境

Matlab 与 Vs 联动

在 Matlab 命令中需要设置 mex -setup
正常情况会搜索出正在使用的 VS 版本，然后找到对应的 xml 文件自行设置，用户只需要选择 使用 C++ 或者 FORIRAN.
然后继续输入 mex -setup C++
如果出现下面错误

请继续输入 mex -setup -v 查看输出中是否有自己使用的编译器版本
如果这里也没有就需要在注册表添加自己的编译器路径，如果有就往下跳打开 regedit ，在 HKLM\SOFTWARE\Wow6432Node\Microsoft\VisualStudio\SxS\VS7 下设置自己 VS 的 VC 目录其中 HKLM\SOFTWARE\Wow6432Node\Microsoft\VisualStudio\ 有可能只到这里，下面两个需要自己创建， SxS,VS7 是项，16.0 是字符值
D:\Programs\vs2019\vs2019\VC
可用在任务管理器找到 VS 的路径，在这个路径里多找找，就能找到类似于
的路径，就到了如果没有这个问题，可以直接下载 xml 文件放到 Matlab\bin\win64\mexopts 文件夹
如果是 VS2019 就使用这两个，其他类似

继续在 Matlab 设置现在就可以使用 mex -setup
每次都选择后面带 C++ 的选项

mex -setup C++, mex -setup C++ -client MBUILD
